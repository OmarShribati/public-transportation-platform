## Project Description
The system is a smart public transportation platform that enables real-time tracking of city buses and their routes. 
It collects live GPS data from buses and combines it with predefined route and stop information to provide passengers with an interactive map experience.  

Users can view all available bus routes from a selected location, see nearby buses in real time, track specific buses, and estimate arrival times.  
The system aims to improve passenger experience, reduce waiting time, and support smarter urban mobility.
